
# OIBSIP Data Analytics Internship – Task 2

## Task: Predicting House Prices with Linear Regression

### Objective
Build a regression model to predict house prices based on features like location, size, and amenities.

### Steps Performed
- Loaded and explored the housing dataset
- Handled missing values and encoded categorical features
- Built a Linear Regression model
- Evaluated the model using MAE, RMSE, and R² score
- Visualized actual vs predicted prices

### Tools Used
Python, Pandas, Scikit-learn, Linear Regression, Matplotlib, Seaborn, Jupyter Notebook

### Outcome
Built a Linear Regression model capable of predicting house prices based on key features, with model performance evaluated using MAE, RMSE, and R² score to measure prediction accuracy.
